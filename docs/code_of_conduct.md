# Code of Conduct

Nu follows the
[Python Software Foundation's Code of Conduct](https://www.python.org/psf/codeofconduct/).
The full text is in
[`CODE_OF_CONDUCT.md`](https://github.com/kovirlabs/nu/blob/master/CODE_OF_CONDUCT.md)
at the repository root.

We want this to be a friendly place: be open, considerate and respectful.
